## Test Backend

### Prerequisites
- Node.js > 8.12
- MongoDB

### Installation

-   make environment
```
$ cp .env.example .env
```
-   install packages
```
$ npm install
```

### Run Project

- seed initial Data
```
$ npm run fresh         // windows
```

- dev development

```
$ npm run dev:window         // windows
```

### Testing Project
```
npm run test:window
```
